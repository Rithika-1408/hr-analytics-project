---

## Project Overview

This project analyses employee attrition patterns and workforce trends using the IBM HR Analytics dataset. It covers the full data pipeline — from raw data cleaning in Python, to SQL-based metric queries, to a 4-page interactive Power BI dashboard.

**Business Problem:** Employee attrition is one of the most expensive challenges for any organisation. Replacing one employee costs 1.5–2x their annual salary. This dashboard helps HR teams identify *who is leaving, why, and from where* — so they can take action before it is too late.

## Prerequisites

### 1. Python
- Python 3.10 or above
- Download: https://www.python.org/downloads/

### 2. Jupyter Notebook
- Comes with Anaconda (recommended) or install separately
- Anaconda download: https://www.anaconda.com/download

### 3. MySQL
- MySQL 8.0 Community Edition (free)
- Download: https://dev.mysql.com/downloads/mysql/
- Also install MySQL Workbench for easy query running

### 4. Power BI Desktop
- Free download (Windows only)
- Download: https://powerbi.microsoft.com/desktop

### 5. Dataset
- IBM HR Analytics Employee Attrition Dataset
- Download free from Kaggle: https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset
- Save the CSV file to: `data/raw/WA_Fn-UseC_-HR-Employee-Attrition.csv`

## How to Run — Step by Step

### Step 1: Install Python dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Run the Python cleaning notebook
```bash
jupyter notebook notebooks/01_data_cleaning_eda.ipynb
```
- Run all cells top to bottom
- This generates `data/processed/hr_cleaned.csv`

### Step 3: Set up MySQL database
Open MySQL Workbench and run these files in order:
```
1. sql/01_create_tables.sql    ← Creates the database and tables
2. sql/02_load_data.sql        ← Loads hr_cleaned.csv into MySQL
3. sql/03_metric_queries.sql   ← Run to verify all KPIs are correct
```

### Step 4: Connect Power BI to MySQL
1. Open `powerbi/HR_Analytics_Dashboard.pbix` in Power BI Desktop
2. Go to Home → Transform Data → Data Source Settings
3. Update the MySQL server to `localhost` and database to `hr_analytics`
4. Enter your MySQL username and password
5. Click Refresh — all visuals will populate

### Step 5: Publish (optional)
1. Sign in to Power BI Desktop with a free Microsoft account
2. Click Home → Publish → Select "My Workspace"
3. Copy the share link for your LinkedIn post and portfolio

## Key Metrics in the Dashboard

- **Overall Attrition Rate** — % of employees who left
- **Attrition by Department** — which team loses the most people
- **Attrition by Age Band** — which age group leaves most
- **Attrition by Salary Band** — does low pay drive attrition?
- **Avg Tenure of Leavers** — how long before people quit
- **Job Satisfaction Score** — satisfaction vs attrition correlation
- **Work-Life Balance vs Attrition** — does burnout drive exits?
